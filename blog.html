Question-1: What are the different ways to select an element in the DOM?

    There are many ways to select an element in DOM.Such as

  • getElementById(): Selects an element by its id attribute.
  • getElementsByClassName(): Selects elements by their class name, returning a live HTMLCollection.
  • getElementsByTagName(): Selects elements by their tag name, returning a live HTMLCollection
  • querySelector(): Selects the first element matching a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

    The key differences is given below:

  • innerHTML: Gets or sets the HTML content inside an element (including tags).
  • innerText: Gets or sets the visible text content, ignoring HTML tags and styles.
  • textContent: Gets or sets all the text content, including hidden text, without HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation involves adding a single event listener to a parent element to handle events for its child elements. It takes advantage of event bubbling to manage events for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and propagates upward through its ancestors.
